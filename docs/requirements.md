# MavlinkProject 需求文档

## 1. 项目概述

MavlinkProject 是一个基于 MAVLink 协议的无人机通信与调度管理系统。该系统提供完整的无人机控制、监控和调度功能，支持多无人机管理、传感器响应、进度链执行等核心功能。系统采用分层架构，包含用户认证、设备认证、Board通信、传感器处理、FRP多中心通信等模块。

### 1.1 项目目标
- 构建一个完整的无人机调度管理平台
- 支持多无人机协同作业
- 实现传感器警报的自动响应
- 提供安全可靠的设备认证系统
- 支持通过 HTTPS 与中央控制服务器通信

### 1.2 术语定义
- **MAVLink**: 一种用于无人机与地面站之间通信的轻量级消息协议
- **Board**: 指无人机、传感器等硬件设备
- **Central**: 中央控制服务器，负责任务链的执行和管理
- **BoardConn**: 后端的设备通信模块，处理与设备的 TCP/UDP 通信
- **Progress Chain**: 进度链，用于定义和执行一系列无人机任务
- **FRP**: 远程端口转发，用于穿透内网，实现远程通信

## 5. 范围限定

### 5.1 功能范围
- 本系统专注于无人机调度管理，不包括前端 UI 开发
- 不包括无人机硬件的设计和制造
- 不包括传感器硬件的设计和制造

### 5.2 技术范围
- 后端使用 Go 语言和 Gin 框架
- 数据库使用 MySQL
- 缓存使用 Redis
- 通信协议包括 TCP/UDP 和 HTTPS
- MAVLink 协议用于无人机通信

### 5.3 性能范围
- 支持最多 100 个设备同时连接
- 支持最多 10 架无人机同时执行任务
- 支持最多 100 个任务链同时存在

- 完整的用户和设备认证系统

- 强大的终端管理功能

该系统能够满足无人机调度管理的各种需求，为无人机的自动化作业提供有力支持。
